# 项目核心知识库

## 项目目标
构建一个高端、互动性强的前端 Engineer 个人作品集网站 (Portfolio)。
风格要求：Deep Dark Navy 背景，玻璃拟态 (Glassmorphism)，动态光效，高级交互。

## 核心共识
- **技术栈**: HTML, Vanilla CSS (Tailwind CSS via CDN for prototyping/speed, or local build), Vanilla JS.
- **设计风格**: 
  - 背景色: #0B0F19 (Deep Dark Navy)
  - 卡片: 玻璃拟态, 悬停发光 (Glow), 局部高亮 (Spotlight).
  - 动效: 平滑过渡, 鼠标交互.
- **语言规范**: 
  - 任务 UI: 简体中文
  - 记忆文件: 简体中文
  - 对话逻辑: 简体中文
  - 专有名词: 英文 (e.g. "Glassmorphism", "Spotlight")

## 排除项
- 避免使用过于简单的配色方案。
- 避免使用复杂的额外框架（如 React/Vue），除非特别必要（目前是纯静态页面）。
